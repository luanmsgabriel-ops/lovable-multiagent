# Lovable Multi-Agent System

> The first multi-agent system for Lovable. 19 specialized skills — orchestrator, memory, critic, improver and more. Send a briefing, get a complete product.

## Quick start

Download `lovable-multiagent-repo.zip`, extract, and install each zip from the `zips/` folder via:

**Settings → Skills → Import → Archive**

Install in this order:
1. `orchestrator.zip`
2. `memory-manager.zip`
3. `critic.zip`
4. `improver.zip`
5. `frontend-developer.zip`

Then send any complex request to Lovable. The orchestrator analyzes, delegates to specialists, passes through critic, and corrects automatically.

## What's inside

19 specialized skills organized in 5 categories:
- **Core** — orchestrator, memory-manager, loop-controller, strategic-advisor, skill-resolver
- **Quality** — critic, improver, bug-detector, opportunity-scanner
- **Build** — architect, database-architect, frontend-developer, backend-developer, api-designer
- **Product** — auth-engineer, billing-engineer, onboarding-engineer, deployment-engineer, product-copywriter
- **Creative** — cinematic-lp
